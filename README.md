## 소개
Open AI 에서 공개한 자연어 처리 모델 gpt2를 활용하여 API로 제공하는  [https://ainize.ai/explore?category=gpt-2](https://ainize.ai/explore?category=gpt-2) 를 활용하여 서비스 기획 및 제작
